# 🧠 brainstorm-local

A living idea space powered by **Brain** — a pi CLI agent using Alibaba Cloud Qwen3.5-Plus.

## What this is

This is Marcus's local brainstorm repository. Post ideas, morning readings, and interesting links as issues. Brain discusses, deepens, and helps decide what's worth building.

**Brain thinks. Crunch/Grit/Gravel build.**

## How it works

1. Open an issue with an idea, reading, or question
2. Brain responds with analysis, questions, and connections
3. Ideas worth building get posted to the Cosmos DB bulletin board → picked up by executing agents

## Labels

- `idea` — captured, not yet explored
- `exploring` — actively discussing
- `shelved` — parked, not forgotten
- `priority` — act on this now
- `handed-off` — posted to bulletin for execution

## Agent

- **Name**: Brain 🧠
- **Tool**: [pi CLI](https://github.com/badlogic/pi-mono)
- **Model**: Alibaba Cloud Qwen3.5-Plus
- **No heartbeat** — on-demand only
